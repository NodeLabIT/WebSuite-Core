Grid-System

Das Grid-System basiert auf den Flex-Attributen von CSS. Die Breite der Container innerhalb einer Reihe ist flexibel, lässt sich jedoch durch Klassen anpassen. Sofern ein der Inhalt in meinem Container mehr Platz benötigt, holt er sich diesen, in Rücksicht auf die anderen.

Foren-Boxen

Der wichtigste Teil eines Forums ist logischerweise die Übersicht der Kategorien. Um einen Container dieser Art zu erstellen ist es wichtig, das Grid-System richtig anzuwenden. Es wird zuerst ein Grid-Parent geöffnet, welcher standardmäßig eine Row enthält. In dieser Row kommt anschließend ein Container, egal mit welcher Größe, mit der Klasse .forums. Anschließend wird der Parent .categorie geöffnet, in welchem sich nun eine weitere Reihe mit den Klassen .noMarginTop und .noMarginBottom befindet. Innerhalb dieser Reihe befindet sich .panel-head. In diesen Container wird anschließend mit der vierten Überschrift der Titel bestimmt. Sofern dies geschehen ist wird .cat-row geöffnet. Innerhalb diesem Container befindet sich .forums-posts, worin anschließend die eigentlichen Boxen mit Titel, Themen und letzten Post zu finden sind.
